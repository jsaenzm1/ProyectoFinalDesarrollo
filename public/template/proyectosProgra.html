Proyecto MongoDB

Este proyecto está creado con la intención de manipular una base de datos online y poder realizar las operaciones básicas como: ingresar, editar y eliminar los registros de una base de datos, esto a través de un formulario que esté conectado con la base de datos. Para este proyecto se ejemplificó la base base de datos para una constructora, la cual permite el ingreso de los materiales de la constructora.

Para este proyecto se trabajó con las siguientes tecnologías.

1. HTML: Por medio de páginas HTML se puede visualizar el formulario que contiene la interfáz gráfica para el usuario.
2. CSS: Utilizado para poder darle estilo a la vista del fomulario.
3. NodeJs: Se utilizó para poder poder comunicar la base de datos con los archivos .js y .ejs, por medio de node se puede accesar a la vista de la base de datos y conectar la base de datos.
4. MongoDB: Es el sistema de base de datos NoSQL, orientado a documentos y de código abierto. En lugar de guardar los datos en tablas, tal y como se hace en las bases de datos relacionales, MongoDB guarda estructuras de datos JSON.


También se utilizaron dependencias para el proyecto, estás facilitaron la comunicación y el manejo de datos entre el usuario y la base de datos. Entre las dependencias están.

1. express: Se utiliza como un marco de aplicación web de back-end para Node.js, está diseñado para crear aplicaciones web y API.
2. ejs: Simplemente significa JavaScript integrado. Se encuentra entre los motores de visualización de plantillas más populares para Node.js y Express.js.
3. mongoose: Es utilizado para poder crear la comunicación con la base de datos de MongoBD.
4. body-parse: Mongoose es una herramienta de modelado de objetos de MongoDB diseñada para funcionar en un entorno asincrónico. Mongoose admite promesas y devoluciones de llamada.
5. dotenv: Se utilizó para poder asiganar un puerto por default.
