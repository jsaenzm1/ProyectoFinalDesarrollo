Tablas de multiplicar con NodeJs

El proyecto consiste en crear una aplicación de consola con NodeJs, donde se impriman las tablas de multiplicar que el usuario desee. La aplicación pedirá la tabla de multiplicar inicial y luego la tabla de multiplicar final, la aplicación creará un archivo .txt de cada una de las tablas. La aplicación tiene que funcionar con el paquete npm “yargs” para poder darle parámetros a la aplicación por medio de la línea de comandos.

Creación del proyecto

1. Crear la carpeta principal que contendrá todos los archivos

Dentro de la carpeta principal.
2. Crear el archivo principal “app.js”
3. Ir a la carpeta principal en consola e iniciar el proyecto: npm init, esto generará el package.json
4. Instalar en consola yargs y colors:

  • npm yargs
  • npm colors

5. Crear el directorio “helpers” y dentro crear el archivo “multiplicar.js”
6. Crear el directorio “config” y dentro crear el archivo “yargs.js”
7. Ejecutar el proyecto con:

  • node app --base=1 --final=8 --listar --hasta=10
  • node app --b=1 --f=8 --l --h=10
  • node app --base=1 --final=8 --hasta=10

base: indica con que tabla de multiplicar queremos empezar
final: indica con que tabla de multiplicar queremos finalizar
listar: indica si queremos imprimir las tablas de multiplicar en consola
hasta: indica hasta que número queremos que lleguen las tablas de multiplicar.
